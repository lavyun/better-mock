# v0.1.3

发布日期：2019-11-29

# Features

  • rurl 支持通配符。Mock.js#177 (opens new window)
  • mock 方法的回调函数增加 headers 字段。 Mock.js#312 (opens new window)、Mock.js#358 (opens new window)

# v0.1.0

发布日期：2019-11-21

# Features

  • 支持对 fetch (opens new window) 的拦截。

# Optimize

  • 优化 Random.address 和 Random.id 对 直筒子市 (opens new window) 的处理。
  • 优化 Random.dataImage 在 Node 端的处理。

# Fix

  • 修复 Random.image 方法不支持带空格的文字。Mock.js#291 (opens new window)

# v0.0.3

发布日期：2019-11-17

# Features

  • 增加 d.ts 声明文件。
  • 新增 Random.phone 方法，生成一个随机的手机号。

# Optimize

  • 优化 Random.dataImage 方法，移除对 node-canvas 的依赖，改为获取远端图片的 buffer，在转为base64。

# Fix

  • 修复无法拦截带参数的 GET，Mock.js#109 (opens new window)，
  • 修复 Mock.mock 方法中, rtype 无需区分大小写。即：get、GET 都可以被成功匹配到。

# v0.0.2

发布日期：2019-11-13

# Features

  • 新增 Random.timestamp 方法 Mock.js#372 (opens new window)，随机生成一个时间戳。
  • 使用 china-location (opens new window) 库作为省市区数据源。
  • 新增 Random.version 方法，随机生成一个版本号。
